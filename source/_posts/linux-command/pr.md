将文本文件转换成适合打印的格式

## 补充说明

**pr命令** 用来将文本文件转换成适合打印的格式，它可以把较大的文件分割成多个页面进行打印，并为每个页面添加标题。

###  语法

```shell
pr(选项)(参数)
```

###  选项

```shell
-h<标题>：为页指定标题；
-l<行数>：指定每页的行数。
```

###  参数

文件：需要转换格式的文件。
