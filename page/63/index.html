特点

ack官网列出了这工具的5大卖点：

  1. 速度非常快,因为它只搜索有意义的东西。
  2. 更友好的搜索，忽略那些不是你源码的东西。
  3. 为源代码搜索而设计，用更少的击键完成任务。
  4. 非常轻便，移植性好。
  5. 免费且开源

File inclusion/exclusion 文件过滤

File finding
