### project

project 可以实现一个工作区内添加多个文件夹，这样就可以查看和修改文件更加的方便 

<!-- more -->

方法：

```
Project -> Sava Project As... 
```

```
Project -> Add Folder to Project...
```

还有切换 Project 的快捷键 ^⌘P
