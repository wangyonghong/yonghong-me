JDK 1.8

接下来看 JDK 1.8

JDK 1.8 中引入了红黑树这个数据结构，当哈希冲突的链表 size 大小超过 TREEIFY_THRESHOLD 的时候，就会变成红黑树，这样查找的速度就会变快。当元素被删除，红黑树节点数量小于 UNTREEIFY_THRESHOLD 的时候，就会转换成链表。
