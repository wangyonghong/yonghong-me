Debian Linux下对软件包索引文件进行排序的工具

## 补充说明

**apt-sortpkgs命令** 是Debian Linux下对软件包索引文件进行排序的简单工具。

###  语法

```shell
apt-sortpkgs(选项)(参数)
```

###  选项

```shell
-s：使用源索引字段排序；
-h：显示帮助信息。
```

###  参数

文件：指定要排序的包含debian包信息的索引文件。
